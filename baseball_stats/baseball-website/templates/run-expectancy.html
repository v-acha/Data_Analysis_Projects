Run Expectancy by Pitch Types

This visualization explores how different teams' run expectancy changes based on the pitch types they face.

It offers an in-depth analysis of how different pitch types impact teams' run expectancy. Users can explore data across various teams and seasons to identify trends, compare performance against specific pitch types, and understand strategic pitching outcomes. This comprehensive tool aids in evaluating team strategies and their effectiveness in different pitching scenarios, providing valuable insights for both fans and analysts.
